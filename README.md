## Docker development environment for Laravel  

## Setup
This setup contains:
+ nginx alpine
+ php-fpm 7.2
+ MariaDB
+ phpMyAdmin


## Pre-requisites

This repository requires [Docker](https://docs.docker.com/) and [Docker Compose](https://docs.docker.com/compose/).




## Installation

~~~~
$ git clone https://github.com/moharami/laravel_lemp.git

$ ./run.sh
~~~~

## View project

Url : http://localhost/
